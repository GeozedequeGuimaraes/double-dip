## Sobre o Projeto

**Double Dip** é um site de uma sorveteria fictícia com catálogo de picolés, desenvolvido com HTML, CSS e JavaScript puro. O projeto conta com busca funcional que filtra produtos em tempo real por nome ou sabor, layout responsivo para desktop e mobile, e uma interface moderna com gradientes e animações suaves.

## Funcionalidades

- **Busca em tempo real** — filtra picolés por nome ou sabor
- **Barra de pesquisa mobile** — expansível ao tocar no ícone
- **Mensagem de estado vazio** — exibida quando nenhum resultado é encontrado
- **Layout responsivo** — desktop, tablet e mobile
- **Cards com hover animado** — elevação e zoom nos produtos
- **Menu sticky** — acompanha o scroll da página
- **Gradientes suaves** — no header e footer

---

## Tecnologias

- **HTML5** — estrutura semântica da página
- **CSS3** — Grid, Flexbox, Media Queries, Gradientes e Transições
- **JavaScript** — filtro de busca e toggle do menu mobile
- **Google Fonts** — Montserrat
